Voice Cursor

System-Wide Dictation with Near Real-Time Precision

❓ The Problem

Standard dictation tools are often frustrating for power users and developers. I identified four critical friction points that Bulbul aims to solve:

  • "The Block Effect": Most tools wait until you finish speaking to "dump" a block of text, ruining creative flow.
  • Window Context Loss: Switching applications often breaks the dictation link, causing lost productivity.
  • Text Duplication: Naive audio chunking leads to repeated words at boundaries.
  • Regional Accent Sensitivity: Global tools often struggle with Indian accents and the "Hinglish" dialect used in technical work.

🛠️ The Build Process

I chose a hybrid approach to combine the ease of web development with the performance of native system access.

Desktop Logic

Electron + Local Go Server for performance.

Audio Core

16kHz PCM mono-channel capture via Web Audio API.

AI Processing

Sarvam AI (Saaras:v3) optimized for local accents.

To achieve the "smooth injection" effect, I bypassed standard paste commands and used Koffi to interface directly with user32.dll, simulating word-by-word typing while strictly locking focus to the target window.

🧠 Technical Deep Dive: DeltaMerger

Deduplication was the biggest challenge. Since audio chunks overlap by 1 second for accuracy, simple insertion causes duplicate text. My custom DeltaMerger solves this elegantly:

// Suffix-Prefix matching logic 1. Maintain a buffer of the last 10 confirmed words. 2. Compare new incoming transcript against the buffer. 3. Identify the longest common suffix/prefix. 4. Extract and inject strictly the unique "Delta".

This ensures a seamless text flow even when the AI model re-evaluates previously spoken words based on new context.

✨ The Impact

Voice Cursor delivers a dictation experience that feels native to the OS:

  • Zero Duplication: seamless text flow across chunk boundaries.
  • Low Latency: Near real-time feedback with less than 1.2s delay.
  • System-Wide Utility: Works perfectly in VS Code, Notepad, and Chrome without losing context.
